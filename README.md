## Overview

The **Developer Productivity Hub** is designed to provide engineering teams and managers with a centralized view of their organization's performance. By emphasizing a clean, intuitive, and responsive UI, this dashboard makes tracking analytics both elegant and efficient.

## Features

- **Developer Directory**: Seamlessly select and view comprehensive details for individual developers, including product ownership and contact email.
- **Team Analytics**: Visual dashboards and charts that deliver actionable insights at a glance.
- **Premium UI/UX**: Built with modern design principles, featuring smooth page transitions and a highly responsive layout tailored for all devices.
- **High Performance**: Powered by Vite for lightning-fast HMR (Hot Module Replacement) and optimized production builds.

## Tech Stack

- **Frontend Framework**: [React](https://react.dev/)
- **Build Toolings**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Architecture**: ES Modules / Component-driven architecture
